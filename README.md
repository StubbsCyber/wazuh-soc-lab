## Overview
This lab documents a SOC-style security monitoring environment using Wazuh SIEM. 
The purpose of this project is to practice log ingestion, alert review, basic triage, 
and incident documentation using Windows endpoints.

This lab reflects hands-on learning in a controlled environment and focuses on 
process, investigation, and documentation rather than production deployment.

## Lab Objectives
- Verify Wazuh SIEM log ingestion from Windows endpoints
- Review and analyze security alerts
- Perform basic alert triage
- Document findings in a SOC-style format

## Environment
- Wazuh SIEM (All-in-One deployment)
- Windows 10 endpoints
- Logs: Windows Security Events, Sysmon (if installed)
